# Install
## Requirements
Python 3.9
## Playwright
https://playwright.dev/python/docs/intro

follow these commands and use the requirements file:
* First install requirements.txt with "playwright" in it
* Then in commandline
```
pip install pytest-playwright
playwright install
```
